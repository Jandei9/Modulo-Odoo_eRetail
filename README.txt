README - Módulo Odoo_eRetail

Este módulo integra Odoo con la plataforma eRetail, permitiendo la sincronización automática de productos, inventario y registros relevantes entre ambos sistemas.

Características principales:
- Sincronización de productos entre Odoo y eRetail.
- Registro de logs de sincronización para auditoría y seguimiento.
- Configuración flexible desde el panel de ajustes de Odoo.
- Automatización de tareas mediante cron jobs.

Este módulo está diseñado para facilitar la gestión centralizada de inventario y productos, optimizando los procesos de integración entre Odoo y eRetail.
